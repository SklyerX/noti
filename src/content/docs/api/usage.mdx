# Usage

Base URL: `https://noti.skylerx.ir/api/notification/usage`

Method: `GET`

Headers:
- `x-api-key`: Your API key

### Request

```bash
curl --location 'https://noti.skylerx.ir/api/notification/usage' \
--header 'x-api-key: <api-key>'
```

### Response

    "data": {

            "usage": 1,
            "remaining": 99,
            "limit": 100
        },
